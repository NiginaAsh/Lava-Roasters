Открытая школа JS основана на изучении языка программирования Java Script, серверной платформы Nod. js и библиотеки React.

Этапи школы

Открытая школа состоит из четырёх уровней:

Level 0

Ближайший курс: 08 мая 2021 г.
Регистрация для всех
  • Изучим HTML - как правильно оформлять структуру строницы
  • Разберём с CSS-селекторами, каскадностью, наследованием
  • На практике рассмотрим вопросы позиционирования элементов, Flexbox
  • Научимся стилизовать формы
  • Поработаем с мультимедиа - изображения, аудио и видео-файлы
  • Изучим основы JS-переменные, циклы, объекты
  • Научимся из JS взаимодействовать с веб- страницей

Учебный курс

Алиф Академия совмесно с ТИКА открыла учебный класс,
куда может прити каждый желающий у кого нет
компьютера или доступа к интернету. Для этого вам
необходима быть участником одного из курсов Открытой
Школы JS. Время посещения класса можно уточнить у
куратора курса.
На данный момент класс функционирует по адресу: г.
Душанбе, ТЦ "Баракат". Ориентир школа № 8,2 этаж.
